Overview

The Contact App is a simple and efficient contact management system that allows users to store, view, and delete contact data. This app integrates with ThunderClient for seamless API requests and provides a user-friendly UI to manage contacts effectively.

Features

  • Add Contacts: Store contact details such as name, email, and phone number using ThunderClient.
  • View Contacts: Display all stored contacts in a well-structured UI.
  • Delete Contacts: Remove any contact easily from the UI with a delete option.

How It Works

Adding a Contact

Use ThunderClient to send a POST request to the backend API with contact details. The backend saves the data in the database and returns a response.

Viewing Contacts

The frontend fetches stored contacts from the backend using a GET request. Contacts are displayed in a list format with essential details.

Deleting a Contact

Users can click on the delete button next to a contact. A DELETE request is sent to the backend, and the contact is removed from the database. The UI updates dynamically to reflect the changes.
